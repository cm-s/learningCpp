Description:

For this project, you will (1) design and (2) and implement a simple adventure game. You can choose any theme for your game you want -- fighting monsters and collecting gold, picking up and eating candy, flying a spaceship to different planets to trade goods, raising animals on a farm, tutoring a student for school -- any theme you want, as long as your game addresses the technical requirements (listed below).

We will share game design ideas and theme ideas together in class, so do not worry if you have not played or created a game like this before.

Your program will, ultimately, be designed and implemented as an object-oriented system. It is assumed that when you start you do not know object-orientation, but do not let that stop you -- you can implement the program as you have implemented your other programs to date, and translate it later into an object-oriented system. We will work together in class so you will know how to do this.

Your program will have the following components:

  • a Player class (in a file Player.py), with the following:
    • a constructor that accepts a name for the player and initializes attributes to reasonable start values
    • a dictionary to store: the player name, points/gold/candy (based on your chosen theme), plus any other player attributes you want to include
    • a list to store items found, or rooms visited, or whatever you choose to store in the list
    • methods to set the various attributes, get the various attributes, add items to the list, view items in the list
    • a method to print all of the values of the player attributes and all of the items in the list
    • other methods as needed to implement your design
  • a Room class (in a file Room.py), with the following:
    • a constructor that accepts a name for the player and initializes attributes to reasonable start values
    • a name
    • a description
    • methods to set the name and description, get the name and description
    • a method representing the primary activity of the room, such as fighting a monster, or solving a puzzle, or playing a game (for example, a number guessing game), or whatever activity is appropriate for your theme
    • other methods as needed to implement your design
  • a Game driver (in a file Game.py; this is where your main menu function will be) that:
    • creates an instance of Player
    • uses a tuple to create a minimum of three (3) instances of Rooms
    • displays the main menu and allows the player to interact with the game

Document all of your Python files using this sample as a guide.

You can find a starter implementation here (you will need all three files -- run Game.py).
